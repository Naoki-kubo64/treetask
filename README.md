# Tree Task Manager (ツリー型タスク管理)

A hierarchical, infinite-canvas task management application built with Next.js and React Flow.
Visualize your tasks as a growing tree, breaking down complex goals into actionable steps.

## Features

- **Infinite Canvas:** Organize tasks spatially without limits.
- **Hierarchical Structure:** Break tasks down into children and siblings (Tab/Enter shortcuts).
- **Drag & Drop:** Reorganize your task tree intuitively.
- **Rich Details:** Add Markdown notes and due dates.
- **Search:** Quickly find tasks with a built-in search filter.
- **Undo/Redo:** Experiment freely with full history support.
- **Local Persistence:** Data is saved automatically to LocalStorage.
- **Export/Import:** Backup and restore your task trees as JSON.

## Visual Skins

Customize your experience with 6 distinct skins:

1. **Refined:** Professional, clean look (Default).
2. **Minimal:** Distraction-free wireframe style.
3. **Nature:** Calming green palette with organic shapes.
4. **Frost:** Dark sci-fi aesthetic with neon accents.
5. **Pixel (Pop):** Retro 8-bit game style with high contrast.
6. **Quest (Map):** Fantasy parchment map style with medieval typography.

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **UI Library:** React Flow, Shadcn/UI, Tailwind CSS
- **State Management:** Zustand
- **Icons:** Lucide React
- **Fonts:** Geist, Press Start 2P, MedievalSharp

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)
